#### Image Generation

### Objective

Generate images from text prompts using a pre-trained Stable Diffusion model.

### Technologies Used

- Python
- Google Colab
- Hugging Face
- Stable Diffusion
- Diffusers

### Sample Prompts

1. A futuristic AI robot working on a laptop
2. A cyberpunk city at night
3. An astronaut riding a horse on Mars
4. A dragon flying over mountains
5. A modern smart city powered by AI

### Results

Images successfully generated using Stable Diffusion.
